## Introduction

 * Using HTML 5 Video API custom video example

 * Using videogular creating a simple video playlist
 
## Setup

#### Install the lib by bower
```
$ bower install
```

#### Run demo
```
# run in borwser
$ ionic serve

# run on emulator
$ ionic platform add ios
$ ionic build ios
$ ionic emulate ios -l

```
